# Sample Terminal Outputs — Password Strength Checker

The outputs below were generated by actually running `password_checker.py`
against 8 sample passwords (see `run_tests.py`). Nothing here is fabricated
— each block is the real console output produced by the program's logic.

Passwords are partially masked in the output (`mask_password()`), showing
only the first character, so it is safe to include this file in a public
repository.
